--------------------
Treasure Hunt Recommendation Server
--------------------
The /treasurehunt directory contains Treasure Hunt's Recommendation RESTful Apache Tomcat server.

The service is hosted on Google Cloud's App Engine, available at this link:
https://alpha-001-dot-treasurehunt-3540e.appspot.com/recommend?username=Alice

The parameter in the GET request can be replaced with other user's username (for instance: Alice, Bob, Claire, mmd).
